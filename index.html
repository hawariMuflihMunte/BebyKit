BebyKit

Introduction

BebyKit is a new Front-end CSS framework which may help you tackle your Front-end development. Rapidly develop your project by using your imagination (free-design mode) or by using framework's built-in Components.

Get started with BebyKit, mobile-first approach and responsive display.

Quick Start

Starter Template

Just copy code below to get quick start front-end development.

HTML 
<!DOCTYPE html>
<html>
    <head>
        <!-- Required Meta -->
        <meta charset="utf-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name=viewport content="width=device-width, initial-scale=1.0">

        <title>Hello BebyKit!</title>

        <!-- Uncompressed BebyKit CSS -->
        <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/hawariMuflihMunte/BebyKit@main/assets/script/css/bebykit.fix.css">

        <!-- Additional 'Dongle' font -->
        <link rel="preconnect" href="https://fonts.googleapis.com">
        <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
        <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Dongle:wght@300;400;700&display=swap">
    </head>
    <body>
        <section class="p-m">
            <h1 class="title">Hello BebyKit!</h1>
        </section>
    </body>
</html>

Download

Use BebyKit by using only 1 css file

via CDN

Import BebyKit CSS file directly from jsDelivr:

In HTML code:

HTML 
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/hawariMuflihMunte/BebyKit@main/assets/script/css/bebykit.fix.css">

In CSS code:

CSS 
@import "https://cdn.jsdelivr.net/gh/hawariMuflihMunte/BebyKit@main/assets/script/css/bebykit.fix.css";

Additional

Font

If you want to use Dongle font from Google Fonts, you can copy this code below and paste right inside your head HTML tag.

HTML 
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Dongle:wght@300;400;700&display=swap">

Please note, if you use Dongle font, then you have to change the --font-size-used variable value.

CSS 
:root {
    --font-size-used: 15pt;
}

This font-size value is integrated with the Dongle font. So you don't have to worry about font-size matters anymore. You can also use --font-size-used, since it is used throughout the BebyKit CSS.

Responsiveness

HTML responsive support for the webpage and framework

Meta Tags

HTML 
<meta name=viewport content="width=device-width, initial-scale=1.0">

Add this code above into your project's head tag to make your project responsive.

Open-source

You can contribute to this project which is open for all designers & developers around the world.
If you want to contribute to this project, you can do that by doing these steps below:

Contribution Guideline

Instruction:

  • The very first, star the project repo.
  • Fork the project repo.
  • Create your own script.
  • Commit & Push your code.
  • Create Pull Request.
  • For developers whose involved convenient, all of the Pull Request will be discussed. If team and other developers are agree with the merging script to the legacy code, then the Pull Request will be triggered after agreement.
  • Congratulations, you have just contributed to an open-source project.
