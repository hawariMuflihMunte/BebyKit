Introduction

Get started to BebyKit CSS Framework

Quick Start

Want to add BebyKit in your project? Use jsDelivr which is free open-source CDN.

CSS

Put this code below in your <head> to get quick start with BebyKit framework

HTML
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/hawariMuflihMunte/BebyKit@main/assets/script/css/bebykit.fix.css">
HTML Setup

Want to get BebyKit quick starter template? You can see example code below

HTML
<!DOCTYPE html> <html> <head> <title>My Project</title> <!-- Import BebyKit --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/hawariMuflihMunte/BebyKit@main/assets/script/css/bebykit.fix.css"> <!-- Responsive Meta --> <meta name="viewport" content="width=device-width, initial-scale=1.0"/> <!-- Additional Common Required Meta --> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <!-- Additional Dongle Font (optional) <link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link rel="stylesheet" href="https://fonts.googleapis.com/css2?family=Dongle:wght@300;400;700&display=swap"> Uncomment to apply Dongle font --> </head> <body> <h1 class="title">Hello World!</h1> </body> </html>
