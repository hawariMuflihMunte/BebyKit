Normal Button

BebyKit's buttons are soft and interactive. Once you pressed it, you can see the state changed to brighter. Indicating that the button was pressed.

HTML 
<a class="btn">button</a>
<a class="btn red-bg">button</a>

Normal Circular Button

This button is worked only on icons. You can't put text inside of it.

HTML 
<a class="btn-circle"><i class="bk-th"></i></a>
<a class="btn-circle red-bg"><i class="bk-th-outline"></i></a>

Outline Button

Outline buttons are cool stuff coming from BebyKit. These buttons background are transparent when they are not being hovered, and change their background color based on text color. You can customize them too!

button button button
HTML 
<a class="btn-outline">button</a>
<a class="btn-outline btn-red">button</a>
<a class="btn-outline btn-black b-n r-l">button</a>
Outline Circular Button

This button is worked only on icons. You can't put text inside of it.

HTML 
<a class="btn-circle-outline"><i class="bk-th"></i></a>
<a class="btn-circle-outline btn-red"><i class="bk-th-outline"></i></a>
<a class="btn-circle-outline btn-green b-n><i class="bk-menu"></i></a>

If the button has background color, then you can apply background colors such as green-bg, red-bg bg-lighter-3 or blue-bg bg-darker-2. But, if the button doesn't have any background colors such as outlined button, then you can't apply background color to its. But you can tckle this problem by using btn-red or btn-white to make the outline and hovered state changed to our styling.

Sizing

BebyKit has its built-in sizes for buttons. you can modify them by using utilities or simply just by following our built-in sizes. You can make a group of buttons by using btn-group class to a div. After done with that, create buttons inside of it by calling btn class name. The sizes for button starts from xs, s, m, l, xl. Look at their differences below:

xs s m l xl
xs s m l xl
xs s m l xl
xs s m l xl

The default size for BebyKit's Buttons is xs size!
xs#1 xs#2

Let's see what happened if we change the font-size inside the buttons...
xs#1 xs#2

The first one, using font-l, second use font-xl, and three use text-m and the last one, four use text-xl.

*Note: To change icon's size, you have to deal with the icon by your own. Set the font-size inside the i class attribute.

The font-sizes can also be changed. By changing font-sizes, it can affect the button's size!

Additional

By using BebyKit's utilities, a modified button can also be created. See examples below:

button button button
HTML 
<a class="p-s r-m light-text font-m red-bg shadow-m unselectable shadow-hover-to-xl">button</a>
<a class="p-s r-m light-bg font-m b-xs shadow-l unselectable shadow-hover-to-m">button</a>
<a class="p-s r-m white-bg bg-enhanced-3 font-m b-s b-orange b-inset shadow-xl unselectable shadow-hover-to-n">button</a>

You can create buttons by your own like these. But, if you prefer to choose this styling way, then you have to set the :hover state, :active state, etc., by your own. And these buttons are also can be selected if user tries to drag text in it. So you have to tackle this problem by using unselectable and also add selectable-none to prevent the texts inside being selected.



We can also make outline button without its outline. Look at examples below: button button button

HTML 
<a class="btn-outline btn-black r-s b-n">button</a>
<a class="btn-outline btn-green r-s b-n">button</a>
<a class="btn-outline btn-red r-s" b-n>button</a>
<a class="btn-outline btn-red b-n"><i class="bk-block font-xl"></i></a>
<a class="btn-outline btn-black b-n"><i class="bk-code font-xl"></i></a>

To apply this, simply add b-n which stand for border-none. This will surely remove any border from buttons.
