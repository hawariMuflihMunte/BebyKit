Button

You can use built-in buttons from BebyKit. Or, you can also create your own button by changing button properties using BebyKit's utilities (e.g. padding, margin, border, radius, etc).

Normal Button

BebyKit's buttons are soft and interactive. Once you pressed it, you can see the state changed to brighter. Indicating that the button was pressed.

button button button
HTML 
<a class="btn">button</a>
<a class="btn red-bg">button</a>

Normal Circular Button

This button is worked only on icons. You can't put text inside of it.

HTML 
<a class="btn-circle"><i class="bk-th"></i></a>
<a class="btn-circle red-bg"><i class="bk-th-outline"></i></a>
<a class="btn-circle light-bg dark-text><i class="bk-menu"></i></a>
Outline Button

Outline buttons are cool stuff coming from BebyKit. These buttons background are transparent when they are not being hovered, and change their background color based on text color. You can customize them too!

button button button
HTML 
<a class="btn-outline">button</a>
<a class="btn-outline btn-red">button</a>
<a class="btn-outline btn-black b-n r-l">button</a>
Outline Circular Button

This button is worked only on icons. You can't put text inside of it.

HTML 
<a class="btn-circle-outline"><i class="bk-th"></i></a>
<a class="btn-circle-outline btn-red"><i class="bk-th-outline"></i></a>
<a class="btn-circle-outline btn-green b-n><i class="bk-menu"></i></a>
